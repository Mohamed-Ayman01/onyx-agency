# Onyx — Web Development Agency Landing Page

Build a premium dark-themed landing page for **Onyx**, a web development agency, using the analyzed design style: dark glassmorphism, purple accent glows, constellation network motifs, and dual serif/sans typography.

## Design System

| Token | Value |
|---|---|
| Background | `#06060B` deep black |
| Surface | `rgba(255,255,255,0.04)` glass panels |
| Primary Accent | `#8B5CF6` → `#A78BFA` violet gradient |
| Secondary Glow | `#7C3AED` → `#DB2777` radial at 10-15% opacity |
| Text Primary | `#FFFFFF` |
| Text Muted | `#9CA3AF` |
| Borders | `rgba(255,255,255,0.08)` |
| Glass Buttons | `rgba(255,255,255,0.08)` + `backdrop-blur(16px)` |
| Headline Font | **Outfit** (bold geometric sans) |
| Accent Font | **Playfair Display** (italic serif for emphasis words) |
| Body Font | **Inter** (clean sans-serif) |

- Tagline pill badge: "We Build Digital Experiences →"

- Subtitle: "Onyx is a web development agency that transforms ideas into stunning, high-performance digital experiences."

### 3. Trusted By (Logo Strip)
- Grayscale client/partner logos scrolling horizontally

### 4. Services
- Section title: "What We **_Deliver_**"
- 3-4 glassmorphic cards with icon, title, and description:
  - **Web Development** — Custom websites & web apps
  - **UI/UX Design** — User-centered design systems
  - **E-Commerce** — Scalable online stores
  - **Mobile Apps** — Cross-platform mobile solutions
- Cards have subtle purple border glow on hover

### 5. Featured Work / Portfolio
- Section title: "Selected **_Projects_**"
- 2-3 project cards with:
  - Large image placeholder area (gradient placeholder)
  - Project name, category tag, brief description
  - "View Case Study →" link

### 6. Process / How We Work
- Section title: "Our **_Process_**"
- 4 numbered steps in a horizontal timeline:
  1. Discovery → 2. Design → 3. Develop → 4. Launch
- Each step has an icon, title, and one-line description
- Connected by a thin line with dot nodes (matching constellation motif)

### 7. Stats / Social Proof

### 8. CTA Section

### 9. Footer
